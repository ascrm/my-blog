## 类型的本质

类型系统不仅仅是为了避免错误，它更是一种表达业务逻辑的方式。

## 高级类型技巧

### 1. 条件类型

```typescript
type MessageOf<T> = T extends { message: unknown } ? T["message"] : never;
```

### 2. 映射类型

```type Record<string, number> = {
  [K in string]: number;
}```

### 3. 模板字面量类型

```typescript
type EventName = `on${Capitalize<string>}`;
```
